## How to make your own

### Build from source

```sh
git clone https://github.com/EverythingSuckz/TG-FileStreamBot -b golang
cd TG-FileStreamBot
go get -u
go build .
chmod +x fsb
./fsb
```

and to stop the program,
 do <kbd>CTRL</kbd>+<kbd>C</kbd>
